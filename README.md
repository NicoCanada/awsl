AWSL
========

在微博自动转发「草」和「awsl」。

![](screenshot.png)

## 安装

### 方式 1：油猴脚本

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 或其它兼容的脚本管理器
2. 打开 [awsl.user.js](https://raw.githubusercontent.com/xingrz/awsl/master/awsl.user.js) 并安装

### 方式 2：Chrome 扩展

1. 将本项目克隆到本地
2. 在扩展管理 (`chrome://extensions/`) 中使用「加载已解压的扩展程序」添加本扩展
